STM32資料 導入編 3

C++の基礎を学んでみよう

STM32では、C++という言語を使ってプログラムを書きます

すべての機能を知っている必要はないが、『関数』と『ポインタ』という重要な概念を説明します

関数とは

関数の概念を説明するために、ここでは部屋の電気をつけてくれるロボットを考えてみよう

関数のイメージ

電気のスイッチを押してくれるロボットがいるとする

このロボットに『どの電気』を『付けるか消すか』を伝えることで

ロボットがスイッチの位置まで移動して電気を付けたり消したりしてくれる

関数はこのロボットのように、必要な情報を伝えて実行するようにお願いすることで様々な処理を実行してくれるものである

またロボットによっては、無事に付けることができたかを報告する機能を持っているものもある

関数の定義

今のロボットをC++のプログラムにするとこのように書くことができる

void Switch(int Place, int Status){

    //ロボットが電気のスイッチを押す処理を書く
}
ロボットの機能 プログラムの文章
void スイッチを付けたかを報告するか(voidの場合はなし）
Switch 関数の名前(他の名前と被らなけばなんでも使える)
int Place 付ける電気の場所 (電気の場所に番号を割り当ててその番号を入力)
int Status 電気を付けるか消すか (0なら消す、1なら付ける)

関数の使い方

関数はこのように使うことができる

関数を使うことで、ロボットの中身の処理を知らなくても、場所と電気を付けるか消すかを決めるだけで動かすことができる

STM32では、色々な難しい処理が関数を使うことで簡単に実行できるようになっている

int main(void){

//場所1の電気をつける(1)
Switch(1, 1);

//場所2の電気をけす(0)
Switch(2, 0);

}
